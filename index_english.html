Version Validation Tool

Category: Professional Project
Python PowerShell JSON

Development and implementation of a tool for validating software versions of the frontend for digital printers.

I completed an internship at Tech Research, a company specializing in solutions for digital printers. The main objective of my internship was to develop and implement a tool for validating software versions of the frontend for these printers. This project aimed to automate and simplify existing tests while integrating new, more advanced tests, all accompanied by a user interface facilitating the use of these tests. The work done during this internship allowed rewriting existing tests in PowerShell* and developing new advanced tests in Python*. A user interface was designed to improve the accessibility and efficiency of the tests. This new tool allows creating, managing, and running test suites at regular intervals, thus facilitating the validation of new software versions for digital printers.
